Información del Listado de Categorías

¿Qué es esta pantalla?

Esta pantalla te permite gestionar las categorías de gastos. Aquí podrás ver, crear, editar y eliminar las diferentes categorías que uses para clasificar tus gastos.

Funcionalidades Principales
  • Nuevo: Botón para crear una nueva categoría.
  • Buscar: Campo de búsqueda para filtrar categorías por nombre o descripción.
  • Volver: Regresa a la pantalla de registro de gastos.
Tabla de Categorías
  • Nombre: Identificador de la categoría.
  • Descripción: Detalle o explicación de la categoría.
  • Acciones: Botones para editar o eliminar categorías (solo disponibles para categorías editables).
Acciones en la Tabla
  • Editar : Modifica una categoría existente.
  • Eliminar : Elimina una categoría del sistema.
Nota:

Algunas categorías del sistema no pueden ser modificadas ni eliminadas ya que son esenciales para el funcionamiento del sistema.
