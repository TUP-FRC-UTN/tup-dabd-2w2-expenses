Información de Lista de Gastos

Vista General

Lista maestro de gastos que permite la gestión integral de registros contables. La vista presenta todos los movimientos ordenados jerárquicamente por fecha, categoría y proveedor para facilitar el análisis y control.

Funcionalidades Principales
  • Exportación de Datos:

    Botones superiores para exportar informes en diferentes formatos

  • Edición de Registros:

    Modificación de gastos individuales mediante el ícono de lápiz

  • Visualización de Documentos:

    Acceso a documentación respaldatoria para auditoría

  • Sistema de Filtros:

    Búsqueda y filtrado por período, categoría, proveedor, tipo y estado

  • Gestión de Gastos:

    Alta de nuevos gastos directamente desde la interfaz

La interfaz está diseñada para optimizar el proceso de gestión y control de gastos, manteniendo la trazabilidad documental requerida para procesos contables.
