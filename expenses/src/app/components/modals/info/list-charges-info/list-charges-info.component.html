Información del Lista de Cargos

Vista General

Lista maestro de cargos que permite la gestión integral de registros contables. La vista presenta todos los movimientos ordenados jerárquicamente por fecha, lote y categoría para facilitar el análisis y control. Cargos que van al lote en el periodo que se cargan, pueden ser de Tipo Multa, Servicios, Limpieza, Reintegro (en caso que se le haga un descuento al lote), Otros, etc;

Funcionalidades Principales
  • Exportación de Datos:

    Botones superiores para exportar informes en diferentes formatos

  • Edición de Registros:

    Modificación de cargos individuales (excepto multas) mediante el ícono de lápiz

  • Eliminación de Registros:

    Eliminación de cargos individuales mediante el ícono de basurero

  • Visualización de Documentos:

    Acceso a documentación respaldatoria para auditoría

  • Sistema de Filtros:

    Búsqueda y filtrado por periodo, categoría, lote y tipo

  • Gestión de Gastos:

    Alta de nuevos cargos (excepto de multas) directamente desde la interfaz

La interfaz está diseñada para optimizar el proceso de gestión y control de cargos, manteniendo la trazabilidad documental requerida para procesos contables.
